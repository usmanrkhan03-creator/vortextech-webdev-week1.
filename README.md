Portfolio Website | Vortex Tech Internship - Week 1

A modern, responsive, dark-themed personal portfolio website built during Week 1 of the software engineering internship at Vortex Tech. This project showcases technical skills, academic background, and professional experiences using clean HTML5 and CSS3.

🚀 Features

* **Smooth Navigation:** Fully functional header menu with anchor links (`scroll-behavior: smooth`) for seamless transitions between sections.
* **Modern Dark Aesthetic:** Clean dark user interface utilizing a sleek black background paired with a sharp coral/red color accent theme.
* **Structured Content:** Organized sections detailing Programming Languages, Software Engineering Core, Education background, and Work Experience.
* **Interactive Design:** Hover animations on action buttons and skill cards for an improved user experience.
* **Responsive Layout:** Optimized to look great on desktop screens, tablets, and mobile devices.

## 🛠️ Technologies Used

* **HTML5:** Semantic markup structure.
* **CSS3:** Custom layout grids, flexbox alignment, and transitions.
* **Font Awesome (v6):** Modern iconography for social platforms and technical skill categories.
